Mit dem Modul Multilabelprint können Sie verschiedene Artikel auswählen und mit einer davor erstellten Layoutvorlage Labels für diese Artikel auf einem PDF Bogen erzeugen. Dazu benötigen Sie auch das Modul Layoutvorlagen.

App Center > Multilabelprint

Anmerkung

Das Modul Multilabelprint gibt es ab Version 17.1 als extra Modul.

Erstellen einer Layoutvorlage für die Multilabels

Wie Sie eine sinnvolle Layoutvorlage für den Multilabelprint erstellen, lesen Sie bitte hier: Layouts erstellen für Multilabelprint

Übersicht der Oberfläche

Artikeltabelle: Hier werden alle Artikel aus den Stammdaten geladen. Sie haben die Möglichkeit Artikel, die Sie auf dem PDF-Bogen haben wollen, zu markieren. Diese Markierung wird auch gespeichert, wenn Sie die Artikelsuche oben rechts betätigen.

Filter: Hier können Sie nur die von Ihnen ausgewählten Artikel anzeigen lassen. Dies ist besonders praktisch wenn Sie Artikel über mehrere Artikelseiten hinweg markieren.

Zurücksetzen Button: Mit diesem Button entfernen Sie alle Haken von markierten Artikel.

Layout: Hier wählen Sie das Layout aus, dass Sie im Modul Layoutvorlagen dafür erstellt haben (siehe oben).

Drucker: Dort wird der Drucker eingestellt, an dem der PDF-Bogen gedruckt werden soll.

Ablauf

  1. Layoutvorlage auswählen - falls noch keine vorhanden: über Layoutvorlagen erstellen

  2. Artikel suchen und auswählen über die Checkboxen.

  3. Mit dem Vorschau Button den PDF-Bogen kurz überprüfen.

  4. Über den Drucken Button ausdrucken.
