Was ist bei Klarna zur Anbindung einzutragen (AB, 10.03.21)
Informationen aus Ticket 106898 von Lisa, da es hier bei Klarna eine Änderung gab

Auf der Klarna Seite findet sich folgende Info unter folgendem Link: https://www.klarna.com/de/verkaeufer/haendlersupport/wo-finde-ich-die-api-credentials/
Gehen Sie zur Einstellungen-App im Händlerportal. Dort können Sie neue API-Credentials erstellen, indem Sie auf “Neue API-Credentials generieren” klicken. Denken Sie daran, die neuen API-Anmeldeinformationen zu speichern, da Sie sie nur einmal sehen können.

Diese API Credentials bestehen aus einem Benutzer (ehemals Merchant-ID) und einem Passwort (ehemals Shared Secret).
Diese API Credentials müssen bei unserer xentral Klarna Oberfläche eingetragen werden.
Hierfür sollten die Felder umbenannt werden in der xentral Klarna Oberfläche.

The credentials consist of two elements:
Username - Consists of your Merchant ID (eid) - a unique number that identifies your e-store, combined with a random string.
Password - a string which is associated with your Merchant ID and is used to authorize use of Klarna's APIs
Note: The API credentials are separate from your Merchant ID, and it is possible to have several API credentials associated with the same Merchant ID.

Beispiel Credentials:

Username: A100001_1b818918f9db Password: rPvw912p34CkdLo4
