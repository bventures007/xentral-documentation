Wenn du dein Passwort vergessen hast, hast du mehrere Möglichkeiten, dein Passwort zurückzusetzen: 

  • Wenn du im Log-In Bereich auf "Passwort vergessen" klickst, wird geprüft, ob in der Benutzeradresse eine E-Mail Adresse hinterlegt ist. Wenn ja, kommt die Zurücksetzungsmail an diese Adresse.
  • Wenn Nein, wird - sofern dein Benutzer Admin-Rechte hat - geprüft, ob in den Grundeinstellungen eine E-Mail Adresse hinterlegt wurde. Falls ja, kommt die Zurücksetzungsmail an diese Adresse.
  • Wenn in beiden Fällen keine E-Mail Adresse hinterlegt ist, wird die Mail an die Adresse geschickt, die bei deiner Lizenz hinterlegt ist.
  • Bitte einen Admin um Hilfe. Admin Benutzer können von allen Nutzern das Passwort ändern.
  • Sollte keiner der oben genannten Wege zum Erfolg führen, wende dich bitte über deinen Key User via Support Ticketsystem an unseren Support, mit der Angabe, dass auch nach über einer Stunde keine E-Mail nach dem Klick auf "Passwort vergessen" angekommen ist
