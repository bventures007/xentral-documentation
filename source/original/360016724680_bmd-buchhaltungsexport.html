Appstore > BMD Buchhaltungsexport

In diesem Modul kann der Export für die Buchhaltung in BMD eingestellt werden. Der Export via BMD ist vor allem für österreichische Kunden interessant.

Übersicht

Der Buchhaltungsexport für BMD Rechnungswesen umfasst die Rechnungen sowie Gutschriften aus einem definierten Zeitraum. Zusätzlich kann über das Projekt gefiltert werden. So spiegeln sich die im System eingerichteten Trennungen von Business Units, Mandanten, Shops oder Kundengruppen auch im Export der Buchhaltungsdaten wider.

Erlöskonten eintragen

Im Reiter Erlöskonten siehst du eine Übersicht über alle angelegten Erlöskonten. Neue Erlöskonten kannst du folgendermaßen eintragen:

  1. Klicke auf +Neues Erlöskonto eintragen im Bereich Aktionen.

  2. Gib in das Feld Konto die Nummer deines Erlöskontos ein.

  3. Optional: Gib eine Bezeichnung ein, um besser nachzuvollziehen wofür das Erlöskonto genutzt wird.

  4. Gib den USt-Steuercode ein. Gibst du hier keinen Wert an, so wird immer der Steuercode "1" für das Erlöskonto exportiert.

  5. Gib den USt-Prozentsatz % an. In Österreicht kann das der Standardsteuersatz von 20% oder ein reduzierter Steuersatz sein.

  6. Klicke auf Speichern.
