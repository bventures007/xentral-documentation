Das US-Steuermodul bietet grundlegende Funktionen für die Bearbeitung von Bestellungen innerhalb der USA. Sie können xentral mitteilen, in welche Länder Sie regelmäßig liefern und die korrekten bundesstaatlichen und lokalen Verkaufssteuern verwenden. xentral wendet dann automatisch die Steuern auf Bestellungen aus diesen Ländern an. Das Modul kann nicht anhand der Lieferadresse ermitteln, welche Umsatzsteuer zu verwenden ist - für diese Funktion ist ein Besteuerungstool wie Avalara erforderlich.

Sie finden das Modul im App Store in der Rubrik Buchhaltung.

Übersicht

Diese Registerkarte zeigt eine Übersicht über die erstellten Ziele für Ihre Sendungen. Sie können ein neues Ziel erstellen, indem Sie auf +Neu klicken.

  • Staat: Name des Staates

  • Inkl. Versand: Gibt an, ob der Versand enthalten ist oder nicht

  • Durchschnittlicher Steuersatz: Steuer auf Bestellungen, die in diesen Staat verkauft werden

  • Ist Nexus: Immer aktiviert, wenn Sie in diesen Staat verkaufen

  • Anzahl der Nexus-Bedingungen: Zählt die Bestellungen in diesen Staat

  • Summe der Nexus-Bedingung: Berechnet den Umsatz der Bestellungen in US-Dollars

  • Ist wirtschaftlicher Nexus: Wird auf "ja" umgeschaltet, sobald Sie einen wirtschaftlichen Nexus für diesen Staat haben. Wenn Sie einen bestimmten Umsatz für Bestellungen in diesem Staat überschreiten, müssen Sie die Steuer anwenden.

Spezielle Steuern

Auf dieser Registerkarte können Sie Sondersteuern erstellen, die für bestimmte Postleitzahlen oder Städte innerhalb eines Staates gelten.

Das Beispiel im Screenshot unten besagt, dass Sie bei einem Verkauf an eine bestimmte Postleitzahl in Gwinnitt County nicht nur die Umsatzsteuer des Bundesstaates Georgia, sondern zusätzlich 2 Prozent erheben müssen. Wenn Sie ein Produkt an jemanden aus Fairbanks in Alaska verkaufen, müssen Sie eine zusätzliche Steuer von 0,5 Prozent erheben.

Es ist möglich, dass die Postleitzahlen innerhalb einer Stadt unterschiedliche Steuersätze haben.

Steuerkennzeichen

Auf dieser Registerkarte werden die Steuerkennzeichen für bestimmte Artikel angezeigt, für die in verschiedenen Bundesstaaten spezifische Steuersätze gelten. Wenn Sie Kleidung an Kunden in New York verkaufen, beträgt der Steuersatz 9,5 Prozent. Nur wenn die Bestellung unter 110 Dollar liegt, ist die Bestellung für Bekleidungsartikel steuerfrei.

Sie können hier neue Steuerkennzeichen anlegen und sie als allgemein steuerfrei oder unter einem bestimmten Wert steuerfrei kennzeichnen.

Protokolle

Auf dieser Registerkarte werden die Staaten erfasst, die aufgrund der Anzahl der Bestellungen und des Umsatzes Ihrer in diesen Staat verkauften Bestellungen zu einem wirtschaftlichen Nexus wurden.

Einstellungen

Auf der Registerkarte "Einstellungen" können Sie dem Modul das Bundesland, aus dem Sie verkaufen, und den Standard-Steuerkennzeichen mitteilen. Außerdem können Sie dem Modul mitteilen, welche Projekte es bei der Anpassung der Steuersätze berücksichtigen soll. Es ist möglich, innere US-Bestellungen zu haben, die von diesem Modul nicht betroffen sind, wenn Sie sie in verschiedene Projekte aufteilen.
