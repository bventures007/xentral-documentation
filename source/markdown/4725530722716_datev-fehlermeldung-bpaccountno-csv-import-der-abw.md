Du kannst die abweichende Debitorennummer auch per.CSV-Import hinzufügen bzw. ändern. Dazu benötigst du die folgenden.CSV-Felder:

```
1:kundennummer;
2:kundennummer_buchhaltung;
```

**Importvorlage**
![](images/?name=inline1865140906.png)

Damit die Kundennummern in Zukunft korrekt angelegt werden, müsstest du die Nummernkreise in den Grundeinstellungen bearbeiten. Hier kannst du festlegen ab welcher Nummern von nun an hochgezählt werden soll, wenn neue Adressen angelegt werden.

*Administration -> Einstellungen -> System -> Grundeinstellungen -> Einstellungen -> Nummernkreise*![](images/?name=inline1292358988.png)
