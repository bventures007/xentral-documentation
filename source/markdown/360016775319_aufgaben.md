UnterAufgabenkönnen Aufgaben erstellt, zugewiesen und bearbeitet werden. Des Weiteren befindet sich hier eine Übersicht mit allen Aufgaben, die dem Mitarbeiter zugewiesen sind.

## Neue Aufgabe

Über+ Neue Aufgabekann eine neue Aufgabe angelegt werden. Diese Informationen kannst du in der "Bearbeiten"-Oberfläche eintragen:

- Aufgabe: Bezeichnung der Aufgabe
- Mitarbeiter: Mitarbeiter, dem diese Aufgabe zugewiesen werden soll
- für Kunde: Sollte die Aufgabe für einen speziellen Kunden zu erledigen sein, kann dieser hier angegeben werden
- Ansprechpartner: Ansprechpartner auf Kundenseite
- Beschreibung: Beschreibung der Aufgabe. Wird eingestellt, dass die Aufgabe auf der Pinnwand erscheint, so wird diese Beschreibung übernommen
- Projekt: Zuweisung der Aufgabe zu einem Projekt
- Teilprojekt: Zuweisung der Aufgabe zu einem Teilprojekt
- Prio: Auswahl, welche Prio die Aufgabe hat
- geplante Dauer in h: geplante Dauer der Aufgabe in Stunden
- Datum/Abgabe bis: Datum, bis wann die Aufgabe erledigt sein muss
- Uhrzeit: Uhrzeit, bis wann die Aufgabe erledigt sein muss
- Regelmäßig (Intervall): Angabe, ob es eine wiederkehrende (täglich, wöchentlich,....) oder einmalige Aufgabe ist
- Zeiterfassung Pflicht: Diese Aufgabe muss zeitlich erfasst werden
- Zeit wird abgerechnet: Angabe, dass die Zeit später abgerechnet werden soll

Diese Informationen kannst du im TabEinstellungeneintragen:

- E-Mail Erinnerung: Einstellung, dass eine Erinnerung an diese Aufgabe per E-Mail erfolgen soll. Voraussetzung hierfür ist ein für den Benutzer eingerichteten E-Mail Account und ein eingerichteter Prozessstarter, der die Aufgabenerinnerungen im Intervall versendet
- E-Mail Anzahl Tage zuvor: Anzahl der Tage vor der Deadline der Aufgabe, an der die E-Mail verschickt werden soll
- Countdown auf Startseite: Anzahl der Tage vor der Deadline der Aufgabe, an der auf der Startseite ein Countdown erscheinen soll
- Öffentlich: Angabe, dass es sich um eine öffentliche Aufgabe handelt, diese ist dann nicht nur für den Benutzer sichtbar
- Auf Startseite: Angabe, dass die Aufgabe auf der Startseite angezeigt werden soll
- Auf Pinnwand: Angabe, dass die Aufgabe auf der Pinnwand angezeigt werden soll
- Status: Status der Aufgabe (offen, in Bearbeitung, abgeschlossen)
- Farbe: Auswahl der Farbe für die Aufgabe auf der Pinnwand
- Pinnwand: Auswahl der Pinnwand, auf der die Aufgabe erscheinen soll
- Notizen: Notizen zu der Aufgabe

Über die grüne SchaltflächeErinnerungsmail sendenkann eine E-Mail Benachrichtigung an den zugewiesenen Mitarbeiter mit allen relevanten Informationen versendet werden.

> **Anmerkung**
>
> Um die Erinnerungsmail automatisch zu versenden, muss im ModulE-Mail-Accountseine E-Mail-Adresse hinterlegt sein, die E-Mails versenden kann. Das heißt bei dieser Adresse muss SMTP aktiviert sein und die entsprechenden Einstellungen ausgefüllt werden (siehehier). Diese Adresse muss mit derStandard E-Mail-Adressein den Grundeinstellungen identisch sein. Weiterhin muss der **Prozessstarter Aufgaben Erinnerung** aktiv sein.

## Aufgabenliste

Unter diesem Tab befindet sich eine Übersicht über die eigenen Aufgaben. Diese können mit dem Filter sortiert und anschließend heruntergeladen werden. Aufgaben können nochmals bearbeitet, erledigt oder gelöscht werden. Zusätzlich kann hier ein Bon für die Aufgabe gedruckt werden.

### Ansicht Mitarbeiter

ÜberAnsicht Mitarbeiterkönnen auch einzelne Mitarbeiter ausgewählt und deren Aufgaben eingesehen werden. Voraussetzung hierfür ist, dass das entsprechende Recht hierfür zuvor gewährt wurde.

### Aufgabenliste als PDF

Es können sowohl eigene als auch externe Aufgaben als Liste im PDF Format heruntergeladen werden.

## Kalender

Im ReiterKalenderwerden die Aufgaben an dem Tag, an dem diese fällig sind, angezeigt. Abgeschlossene Aufgaben werden mit einem Haken markiert. Aufgaben können direkt im Kalender überarbeitet und verschoben werden. Die Kalender-Ansicht ermöglicht dem Team-Leiter eine Planung einzelner kleinerer Aufgaben oder die Planung über längere Zeiträume hinweg.

## Zeiterfassung für Aufgaben

In der Aufgabe können die OptionenZeiterfassung PflichtundZeit wird abgerechnetfür die Zeiterfassung markiert werden. Werden die beiden Haken gesetzt, so wird die Zeit automatisch mit Abschluss der Aufgabe gebucht. Wenn du hier eine Uhrzeit angibst, wird die Zeit vom angegebenen Zeitpunkt bis zum Zeitpunkt des Abschlusses der Aufgabe gebucht. Ohne Zeitangabe wird 00:00 Uhr als Startzeit verwendet. Der Name der Aufgabe wird als Tätigkeit im Eintrag der Zeiterfassung übernommen. Änderungen können über den ReiterEigene Zeiterfassung Übersichtvorgenommen werden.
