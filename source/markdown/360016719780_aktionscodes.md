Mit dem Modul "Aktionscodes" erscheint ein zusätzliches Feld in Belegen, welches danach ausgewertet werden kann. Codes können schnell angelegt werden und Belege wie Angebote, Aufträge etc. können einfach auf diese Aktionen gebucht werden. Über eine Statistik können so die Umsätze der Aktionen in selbst definierten Zeiten ausgewertet werden.

App Center → Aktionscodes

## Erstellen von Aktionscodes

Unter App Center → Aktionscodes → NEU können die Aktionscodes eingegeben werden.

## Aktionscode im Beleg

Im Beleg gibt es unterhalb des Adressfelds das Aktionscode-Feld, in welches der Aktionscode eingegeben wird.

## Auswertung der Aktionscodes

Unter Controlling → Aktionscodes können diese Codes in den Belegen aufgelistet werden.

### Auswertung über alle Belege

Über den Button "Aktionscodes auswerten" findet eine Auswertung über alle Belege statt.

### Auflistung der Anzahl pro Beleg

In den anderen Tabs kann die Anzahl der einzelnen Aktionscodes pro Beleg dargestellt werden.
