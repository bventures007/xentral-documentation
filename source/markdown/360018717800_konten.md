# Parameter (PA, 08.01.21)

Seit der Version 20.3 stehen über die Oberfläche neue Banken-Parameter zur Verfügung. Hierzu muss der Wizard erfolgreich ausgefüllt werden. Voraussetzung hierzu ist eine valide IBAN + BIC.

API_MAX_DAYS_BEFORE=>30; Gibt in diesem Beispiel alle Zahlungseingänge der letzten 30 Tage zurück. Ist wichtig bei Banken, die keine zu großen Abstände vom heutigen Tag zulassen. Bei den meisten Banken ist hier bei 90 Tagen Schluss.

Theoretisch sollte auch API_DAYS so funktionieren, aktuell aber nicht gesichert.

API_DAYS_WAIT=>1; Dadurch wird der Bis-Wert bei der Abholung reduziert. In Kombination mit den o.g. Parametern können hier nun einzelne Zeiträume abgeholt werden.

FHP_LOGGING=>5; Gibt bei aktiviertem logfile (module=logfile) die Transaktionen beim Abholen der Zahlungseingänge ab. Hier können die Detailstufen 1-5 verwendet werden.

FHP_TAN_VERFAHREN=>PIN-TAN; Hier kann das verwendete TAN-Verfahren direkt vorgegeben werden. Die Bezeichnung hängt hier von der jeweiligen Bank ab. Theoretischs sollten die möglichen Bezeichnungen über das LOGGING erfahren werden.

## Beispiel: Abfrage eines bestimmten Zeitraums

Heute ist ein Freitag.

API_MAX_DAYS_BEFORE=>5;

API_DAYS_WAIT=>2;

-> Holt die die Zahlungen der Tage für Montag, Dienstag und Mittwoch in dieser Woche ab.
