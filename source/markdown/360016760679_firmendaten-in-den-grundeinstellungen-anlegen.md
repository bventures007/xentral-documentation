Um die richtige Kommunikation mit Kunden, Lieferanten usw. zu gewährleisten, benötigtXentral deine vollständige, aktuelle Adresse. Die Firmendaten findest du in den Grundeinstellungen für das XentralSystemsetup.

> **Anmerkung**
>
> Bitte stelle sicher, die wichtigsten Daten einzutragen, damit dir in anderen Funktionen die Daten nicht fehlen, z.B. benötigst du das ISO Länderkürzel für das Land deiner Firmenniederlassung für Lagerort, Versand und Länderkonfigurationen. Dein Firmenname wird auch innerhalb von Xentral im System für dein Team angezeigt.

InXentral NextGen: **Einstellungen > Dein System einrichten > Firmendaten ** Im Classic Design: **Administration > System > Grundeinstellungen ** Alternativ über die Systemsuche: **Grundeinstellungen **>** Anmerkung**
>
> Eine Übersicht der Einstellungen in Xentral NextGenfindest du indiesem Artikel.

Dazu kannst du im Reiter **Einstellungen **, Unterreiter ** Firmenanschrift**die Firmenanschrift des Hauptsitzes hinterlegen, sowie die Steuernummer bzw. die Umsatzsteuer-ID der Firma eintragen:

| Feldbezeichnung | Beschreibung |
| --- | --- |
| **Name*** | Name deiner Firma (Hauptsitz, rechtlicher Name) |
| **Straße*** | Straße und Hausnummer deines Firmensitzes |
| **PLZ*** | Postleitzahl deines Firmensitzes |
| **Ort*** | Ort deines Firmensitzes |
| **Land*** | ISO Länderkürzel, wird benötigt für länderspezifische Einstellungen (hier z.B. DE für Deutschland und AT für Österreich) |
| **USTID bzw. Steuernummer*** | Umsatzsteuernummer deiner Firma |
| SEPA Gläubiger ID | SEPA Gläubiger ID (wenn du das Modul Lastschrifteinzug verwenden möchtest, ist die SEPA Gläubiger ID rechtliche Voraussetzung für deinen Bankeinzug) |
| EORI Nummer | Diese Identifikationsnummer ist für alle Unternehmen notwendig, die Güter in die EU importieren oder aus der EU exportieren. Die EORI-Nummer erhältst du von deiner nationalen Zollbehörde. |

> **Anmerkung**
>
> Die Grundeinstellungen gelten systemweit. Verfeinerte Einstellungen, beispielsweise auf Projektebene, können je nach Workflow-/Projekteinstellung vor den Grundeinstellungen Vorrang haben. Bitte beachte dies in den jeweiligen Einstellungen z.B. E-Mail Vorlagen oder Mail-Adressen, die Projekt- oder Saleschannel-bezogen eingestellt werden.
