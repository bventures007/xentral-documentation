Der Lieferschein ist ein Warenbegleitpapier, in welchem Details über die Warenlieferung enthalten sind. In xentral kann im Modul **Lieferschein** eine Übersicht über alle Lieferscheine eingesehen sowie neue Lieferscheine erstellt werden. Außerdem können die Lieferscheine als CSV exportiert werden.

Besonders relevant ist dieses Modul, wenn mit Lieferscheinen gearbeitet wird.

## Übersicht

Die **Übersicht ** zeigt eine Auflistung aller Lieferscheine. Über den Reiter**in Bearbeitung** kann auch eine Liste aller Lieferscheine angezeigt werden, die sich in Bearbeitung befinden.

## Neuer Lieferschein

Es bestehen zwei Möglichkeiten, wie ein neuer Lieferschein erstellt werden kann. Zum einen kann ein Auftrag über das **Aktions ** menü als Lieferschein weitergeführt werden, zum anderen kann ein neuer Lieferschein über den Button**+NEU** erstellt werden.

## Detailansicht

Über das **Stift**

-Icon kann in die Detailansicht eines bestimmten Lieferscheins gesprungen werden. Wurde der Lieferschein bereits versendet, können die Details nicht mehr verändert werden. Wurde der Lieferschein noch nicht versendet bzw. freigegeben, können Details angepasst oder ergänzt werden. Die Details enthalten alle wichtigen Informationen eines Lieferscheins

### Positionen

Unter diesem Reiter können die einzelnen Bestellpositionen angegeben werden.

### Vorschau

Unter diesem Reiter wird eine Vorschau des Lieferscheins erstellt.

### Protokoll

Unter diesem Reiter befindet sich eine Übersicht über Belege, welche direkt über PDF eingesehen werden können. Zudem ist hier eine Auflistung der Artikel, sowie die Informationen wer wann was mit dem Lieferschein gemacht hat und ein PDF-Archiv.

Wenn die Belege bzw. hier im Konkreten Positionen in Lieferscheinen ein- bzw. ausgelagert werden soll, dann kann die Funktion[Belege ein- und auslagern](https://help.xentral.com/hc/de/articles/360016721280#UUID-456bb23f-a0d1-04ac-9ad3-db7c739a6535)genutzt werden.

## Paketmarke

In diesem Reiter können Paketmarken erstellt und gedruckt werden. Dabei werden die Informationen für den Empfänger von den Angaben unter **Details** übernommen.

- **Gewicht:** Gewicht der Lieferung
- **Nachnahme:** Lieferung erfolgt per Nachnahme
- **Extra Versicherung:** Angabe, dass eine extra Versicherung besteht
- **Leitcodierung:** Angabe, dass eine Leitcodierung vorhanden ist
- **Abholdatum:** Angabe des Abholdatums
- **Wunschtermin:** Angabe des Wunschtermins der Lieferung
- **Retourenlabel drucken:** Auswahl, dass ein Retourenlabel gedruckt werden soll

Im Anschluss kann die Paketmarke entweder gedruckt oder an dieser Stelle nochmal eine andere Versandart ausgewählt werden.

## Seriennummern

In diesem Reiter können Angaben zur Seriennummer für die einzelnen Artikel getätigt werden. Dies kann sowohl manuell, als auch über den Assistenten erfolgen.

## Dateien

Unter diesem Reiter können alle hochgeladenen Dateien eingesehen sowie neue Dateien angelegt werden (z.B. Lieferschein). Diese Dateien können exportiert werden.

## Manuelles Ein-/Auslagern

In diesem Reiter können die einzelnen Artikel manuell ein- bzw. ausgelagert werden.

## Bewegungen

In diesem Reiter liegt eine Übersicht über alle Bewegungen der Lieferartikel.

## Export des Lieferscheins

Es gibt die Möglichkeit neben Auftrag, Angebot, Rechnung und Gutschrift auch den Lieferschein als CSV-Datei zu exportieren.

Dazu wird der jeweilige Lieferschein über das Mini-Detail (Pfeil auf der linken Seite) geöffnet. Der Export des Lieferscheins kann dann über das **Aktionsmenü** ausgelöst werden. Dabei wird eine Datei erzeugt, die sich im Anschluss bei einem anderen xentral-System über den Beleg Importer einspielen lässt.

## FAQ

### Wie gehe ich vor, wenn die Funktion "Lieferschein auslagern" bei manchen Lieferscheinen nicht funktioniert?

Damit diese Funktion immer funktioniert, musst du folgende Punkte erfüllen:

- Der Artikel ist ein Lagerartikel. Andernfalls kannst du ihn überhaupt nicht auslagern
- Der Lieferschein ist in einem Projekt angelegt, so dass du den Lieferschein auslagern kannst
- Der Lieferschein wurde noch nicht ausgelagert
