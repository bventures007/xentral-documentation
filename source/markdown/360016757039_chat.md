## Voraussetzungen

- Mitarbeiter benötigt Rechte für Chat
- Mitarbeiter muss in der Adresse auch die Rolle „Mitarbeiter“ vergeben bekommen haben.

## Prio Nachricht

Mit demPrio Hakenunten rechts können Sie eine Nachricht mit Prio verschicken. Dadurch bekommt der Mitarbeiter, an den die Nachricht gesendet wird, auch ein kleines Popup im Browser.

![Chat_priority.png](https://help.xentral.com/hc/article_attachments/5010450866844)

## E-Mail Benachrichtigung von ungelesenen Nachrichten

E-Mail Benachrichtigungen über das Vorliegen einer ungelesenen Chat Nachricht werden automatisch von xentral an die E-Mail Adresse eines Benutzers (Stammdaten) versendet, wenn der User nicht eingeloggt war.

## Chat Prozessstarter

Folgender Prozessstarter muss aktiviert sein:

*Administration > Einstellungen > System > Prozessstarter > chat*

![Chat_process_starter.png](https://help.xentral.com/hc/article_attachments/5010421540380)

## Chat Benachrichtigung pro Benutzer

Die Chat Benachrichtigung kann pro Benutzeraccount deaktiviert werden:

*Administration > Einstellungen > System > Benutzer > entsprechenden Benutzer editieren:*

- Im Kalender/Chat ausblenden > aktivieren, um Benutzer im Chat und im Kalender auszublenden
