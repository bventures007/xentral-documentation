## KNO/BW: Import von Eigenschaftsübersetzungen

Die CSV-Definition ist wie folgt:
1:nummer;
2:eigenschaftnameeindeutig1;
3:eigenschaftwert1;
4:eigenschaftname1_fr;
5:eigenschaftwert1_fr;
6:eigenschaftname1_en;
7:eigenschaftwert1_en;
![](https://user-images.githubusercontent.com/50359667/101047671-4e438300-3582-11eb-9eb6-245b1c690ea7.png)

Die csv-Datei muss wie folgt erstellt werden:
![](https://user-images.githubusercontent.com/50359667/101047860-76cb7d00-3582-11eb-9377-af5d656b6969.png)

Eine solche Zeile würde ergänzt werden.

Wird Zeile 2 wie folgt geändert: 2:eigenschaftname1; dann wird der bestehende Wert nicht überschrieben.
Ergebnis unter index.php?module=artikel&action=eigenschaften&id=[ARTILCEID] wie folgt:
