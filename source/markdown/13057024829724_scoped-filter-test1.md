### Release Version 23.0.21

*Release date: 17-April-2024 *#### Bug Fixes **Back Button ** Several issues with the Back button in Xentral NextGen have been fixed. (PIM-1168)**Error Message on Batch Adding ** An error message that appeared when adding a batch to production has been fixed. (FFU-1630)**Error Message on Product Overview Tab ** An error message that appeared when users tried to open the product overview tab in the Order Proposal module has been fixed. (FFU-1623)**Amazon Prime Shipping Method ** Fields to connect the Amazon Prime shipping method via Amazon MWS have been removed as this method of connection is no longer functional. (FFU-1452)**Broken Filter in Invoice Overview ** A broken filter in the invoice overview screen has been fixed. (FAC-1819)**Update Positions Button**

The Update positions button no longer affects items marked as postage to prevent issues with postage items that were changed by the Split shipping module. (FAC-1007)

#### Improvements **Shipping Method Settings ** Settings for the shipping methods GLS, DPD, and DHL (alt) have been divided into basic and expert settings. (FFU-1015)**Copying Shipping Method ** Copying a shipping method now generates a unique entry for type to avoid problems with duplicate types. (FFU-662)**Error Messages in Datev Connect**

Error messages in Datev Connect have been improved for cases when the login fails. (FAC-1842)

#### New Features **API Endpoint for Documents ** An API endpoint that delivers all related documents for a specific credit note is now available in Beta. (FAC-1824)**Mapping Product Listings**

You can now map all your product listings in Kaufland with one single table based on EAN codes. (ECOM-1665)
