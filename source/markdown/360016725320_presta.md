## Presta Schnittstelle - Installation & Einrichtung

Mit der Presta Schnittstelle können Sie Xentral mit Presta verbinden. Die Presta Schnittstelle gibt es in unserem Shop.

## Features der Schnittstelle

- Aufträge aus Presta abholen
- Auftragsstatus bei Versand umstellen inkl. Meldung Trackingnummer
- Artikel einzeln von Xentral nach Presta oder von Presta nach Xentral übertragen
- Meldung aktueller Lagerzahlen bei Bestandsänderungen

Die Rechnung und oder Lieferschein kann durch Xentral mit eigenen Nummernkreisen erstellt werden. Die Belege werden im Versandprozess automatisch erstellt und versendet bzw. am Versanddrucker ausgedruckt.

### Shopschnittstelle in Xentral anlegen

1. Shopschnittstelle anlegen
  Unter Administration → Shop Schnittstelle → Übersicht → NEU kann man nach Presta suchen

1. Presta Daten eingeben
  Vergeben Sie einen Shop-Namen und stellen Sie fürs Erste die Einstellungen wie im Screenshot ein.

  Danach tragen Sie bitte den Schlüssel in das Feld API Key ein. Die Shop-URL muss mit dem API-Verzeichnis enden:[http://meineseite/api/](http://meineseite/api/)

  Die Shop-ID im Subshop Mapping wird immer dann gesetzt, wenn eine Bestellung aus dem jeweiligen Subshop importiert wird. Damit kann auf einfache Weise ein Subshop Mapping aufgebaut werden. Die ID für den Subshop konfiguriert die Shop-ID für die Meldung des Lagerbestandes.

  Im Bereich Steuergruppenmapping müssen die Steuersätze aus Xentral auf die Steuerklassenobjekte aus Presta gemappt werden. Diese finden Sie unter Optimierung → International → Steuersätze → Steuerregeln.

1. Verbindung prüfen
  Am Schluss überprüfen Sie mit einem Klick auf "Verbindung prüfen" ob es geklappt hat:Sollte es zu einem Fehler kommen überprüfen Sie bitte Folgendes:- Die Daten aus Presta wurden 1:1 so in Xentral hinterlegt

  - Die URL wurde richtig mit /api/ am Schluss angegeben
1. Weitere Einstellungen vornehmen
  Wie Sie anderen Einstellungen vornehmen, erfahren Sie in[diesem Helpdeskeintrag](https://xentral.com/helpdesk/kurzanleitung-einrichtung-online-shop-schnittstelle).
